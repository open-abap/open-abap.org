open-abap

By transpiling ABAP to Javascript, ABAP can run on Node.js.

open-abap provides various libraries for running transpiled code, plus general purpose multi target utilities.

  • transpiler
  • runtime
  • database-sqlite
  • database-pg
  • database-snowflake

Runtime

ABAP on Node.js
  • open-abap-core, Core ABAP artifacts
  • open-abap-bal, Application Logging
  • open-abap-deprecated, Deprecated artifacts
  • open-abap-gui, GUI controls
  • open-abap-fs, Filesystem bindings
  • open-abap-gpio, Raspberry Pi GPIO bindings

General purpose, multi target

Multi-target ABAP libraries. Run on ECC + Steampunk + Node.js
  • open-abap-ssh, SSH client in ABAP
  • open-abap-pdf, Standalone PDF generation
  • open-table-maintenance, Low footprint ABAP table maintenance
  • cross-http-client, Cross platform http client
  • cross-http-server, Cross platform http server
  • open-abap-odata, OData/Gateway shims for Node.js and Steampunk

Shims

Testing and Continuous Integration
  • express-icf-shim, Shim for running ICF services on Express
  • express-steampunk-shim, Shim for running Steampunk HTTP bindings on Express
  • browser-icf-shim, Run in browser only without backend
  • shims-rfc, Shims for tRFC + qRFC + bgRFC
  • lambda-icf-shim, AWS Lambda ICF shim
